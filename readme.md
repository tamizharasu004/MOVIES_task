#MOVIES 

This is a Java JDBC project to manage movies.

## Database
Database Name: movi  
Table Name: movies

## Table Structure
- id (INT, Primary Key, NOT NULL)
- movie_name (VARCHAR(150), NOT NULL)
- rating (INT)
- year (INT)

## Features
- Add Movie
- View Movies
- Update Movie
- Delete Movie

## Technologies Used
- Java
- JDBC
- MySQL
- Git
